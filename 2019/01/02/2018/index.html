2018

2019-01-02 Fragments

2014年的现在应该紧张准备高考。
2014年的9月正式入学。
2018年的6月正式毕业。
2018年的9月再次入学。
转眼，2018年也过去了。

Preface

我后来才慢慢认可，仪式感是一种很重要的东西。

Flow

1月，1月的时候在学校待到很晚才回家，老宿舍冷冷的，我有好几天都懒在被窝里到下午。
2月，寒假在家写了一个书签主页info.x-fei.me，基于Laravel5。
3月，先回学校待了半个月，之后来SYSU，一边学习一边做毕设。
4月，完成毕设，入坑tensorflow，准备Google Summer Code(准备太不充分，吹了)。
5月，回CSU准备答辩，PUBG。
6月，答辩，拍毕业照，毕业拨穗乌龙（反向给教授拨穗）。
7月，在SYSU参加MICCAI2018年SKIN LESION的比赛，学习Deeplabv3+，成为Tensorflow的Contributor（只是更新了部分文档）。
8月，因为太菜所以成绩不佳。回长沙，借住在海洋家一周，终于拿到驾照（感谢海洋🍻）。回家待了两周，开始刷题。
9月，开学以及确定课题方向，刷了满满一个月的题。
10月，月初的时候大圣来找我玩了几天。整个月都在实验、上课，和霖哥一起做了一个外包，写后端。
11月，实验、上课，写外包，修复bug，重构代码，入坑Python爬虫，周末游戏。
12月，准备投稿，“逃课”2周，17号在导师的帮助下，投出第一篇。之后有所懈怠。

Reward

Diary

1月到3月的时候，开始写日记，坚持写了66天，都发在了公众号上。
问题是，我并不是一个合格的内容运营者，单纯的日记，也没有必要发在公众号上，缺少时间的积累、沉淀。

Leetcode

200道题，完整刷了9周，70天左右。主要在8月与9月的时候，之后由于懒且事多，且菜，所以停了。

Blog

下半年久坐在实验室当中，有时候事情虽然多，但是也想写写总结，于是下半年高产，预期19年一年依然高产。

Memory

研究生入学，受同门嘉鑫影响比较大，生活多了很多乐趣（喝了很多奶茶）。感谢我的同门以及师兄师姐师弟师妹。

B

本学期上完研究生的所有课程，但这不是我人生的最后课堂。我越发觉得自己和现在的课堂格格不入了，幸运的是我有一套自己的学习方式，但是并没有好好地去总结它，因此，之后需要注意的就是总结自己的学习方式，调试它，最优化它。
